Get Involved!

Curious Coders is designed for coders of all levels of ability to learn about contributing to open-source projects and experiment.

Just signin/register at Github and follow the link below to get started.

Github

    Clone the Directory

    (Make sure you have git and npm/node installed)

  1. Click 'clone or download' and copy the URL
  2. Use the cd (change directory) command in your terminal/CLI to navigate to your desired location.
  3. Then type 'git clone' followed by your copied URL.

    4. Install dependencies

  4. cd into the directory you just created and type 'npm install' to install the necessary dependencies.
